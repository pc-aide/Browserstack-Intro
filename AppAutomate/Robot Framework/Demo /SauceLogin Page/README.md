# SauceLogin Page

---

## Install

### Robot Framework
````powershell
pip install robotframework
````

### browserstack-sdk
````powershell
pip install browserstack-sdk
````

### lib appium robot framework
````powershell
pip install robotframework-appiumlibrary
````

---

## Deploy to Automate
````powershell
browserstack-sdk robot test_sauceLoginPage.robot
````
